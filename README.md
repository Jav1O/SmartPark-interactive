# SmartPark Interactive

Interactive web project focused on improving the parking search experience through a smart and user-centered interface.

The repository includes different interactive prototypes and exercises using JavaScript, HTML, CSS and browser APIs, exploring concepts such as user interaction, webcam input, speech/motion interaction, sentiment analysis, weather data and real-time interface feedback.

## Main Topics

- JavaScript-based interaction
- HTML and CSS interfaces
- Browser APIs
- Interactive prototypes
- User-centered design
- Smart parking concept

## Repository Structure

- `solucionesEjerciciosWebAPI/`: exercises using Web APIs.
- `solucionS7clima/`: weather-related interface functionality.
- `solucionS7sentimientos/`: sentiment-related interaction.
- `solMando/`: controller-based interaction.
- `solucion_practicaSpeechMotion/`: speech and motion interaction.
- `EL PROBLEMA DE ENCONTRAR PARKING RÁPIDAMENTE MIENTRAS CONDUCIMOS.txt`: project context and problem statement.

## What I Learned

This project helped me improve my understanding of interactive web development, browser APIs, user-centered design and practical JavaScript programming for real-world interface problems.
